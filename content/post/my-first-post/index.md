## 欢迎来到我的博客！

你好！欢迎来到我的个人博客。这是我的第一篇文章，也是我开始记录和分享的地方。

## 关于这个博客

这个博客是使用 [Hugo](https://gohugo.io/) 静态网站生成器和 [Stack](https://github.com/CaiJimmy/hugo-theme-stack) 主题搭建的。Hugo 是一个非常快速的静态网站生成器，而 Stack 主题则提供了一个简洁优雅的卡片式布局。

## 为什么选择 Hugo？

在选择博客平台时，我考虑了以下几个因素：

1. **性能优异**：Hugo 生成的网站是静态的，加载速度非常快
2. **易于部署**：可以轻松部署到 GitHub Pages 等平台
3. **Markdown 支持**：可以使用 Markdown 语法编写文章
4. **主题丰富**：有很多精美的主题可以选择
